**Role:** You are a professional trivia researcher. Your goal is to create
high-quality multiple-choice questions that are fun, fair, and have exactly one
objectively correct answer.

**Task:** Generate {count} trivia questions for the category **"{topic}"**.

**Category description:** {description}

**Example question for this category (use as a style guide, do NOT repeat it):**
{example}

Distribute the questions across difficulties: roughly 40% Easy, 40% Medium,
20% Hard.

**Output:** Return ONLY a JSON array. No markdown fences, no commentary.
Each element must match this schema exactly:

```json
[
  {{
    "question": "A concise question (one sentence, under 120 characters preferred)",
    "answer": "The correct answer (short, commonly known form)",
    "alternative_answers": ["Other valid phrasings or synonyms for the answer"],
    "distractors": ["Wrong 1", "Wrong 2", "Wrong 3"],
    "difficulty": "easy | medium | hard",
    "source": "A specific, verifiable reference — preferably a URL to a reputable source. If no URL is available, cite a specific book (title + author + page/chapter) or a named report/publication. Avoid vague attributions like just an organization name.",
    "clarity_note": "Brief note explaining why the answer is unambiguously correct, or why alternatives are also accepted"
  }}
]
```

- `alternative_answers`: Other correct or widely accepted answers (e.g.,
  "Peking" for "Beijing"). May be an empty array if there is only one
  valid answer.
- `clarity_note`: A short explanation (one or two sentences) that a human
  reviewer can use to verify the question. Not shown in-game.

**Difficulty Definitions:**
- **Easy:** Common knowledge most adults would know.
- **Medium:** Requires some familiarity with the topic.
- **Hard:** Requires specific or expert-level knowledge.

**Rules:**
1. Each question must have exactly ONE correct answer and THREE plausible
   distractors of the same type (e.g., if the answer is a person, all
   distractors must be people from a similar era or field).
2. Keep questions concise — this is a fast-paced game with a 30-second timer.
3. Avoid ambiguous phrasing. Do not use "Which of these..." since button
   order is randomised.
4. No duplicate or near-duplicate questions within the batch.
5. Distractors must be definitively wrong, not "arguably correct."
6. Do not include questions whose answers change over time (e.g., "Who is
   the current president?").
7. If a question could have multiple valid answers, list the primary one in
   `answer` and the rest in `alternative_answers`. Ensure none of the
   `alternative_answers` appear as distractors.

**Example output:**
[
  {{
    "question": "What is the brightest star in the night sky?",
    "answer": "Sirius",
    "alternative_answers": ["Alpha Canis Majoris", "Dog Star"],
    "distractors": ["Betelgeuse", "Polaris", "Rigel"],
    "difficulty": "medium",
    "source": "https://science.nasa.gov/star/sirius/",
    "clarity_note": "Sirius (Alpha Canis Majoris) has an apparent magnitude of −1.46, making it the brightest star visible from Earth."
  }}
]
